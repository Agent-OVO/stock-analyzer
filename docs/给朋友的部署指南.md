# 给朋友的部署指南

感谢您对 A股自选股智能分析系统的关注！

本文档将帮助您快速部署并配置系统。

---

## 第一步：获取系统代码

### 方式一：从GitHub克隆（推荐）

```bash
# 安装Git后执行
git clone <项目地址>
cd 股票智能分析系统
```

### 方式二：接收压缩包

1. 解压到任意目录，如 `D:\股票智能分析系统`
2. 确保目录结构完整

---

## 第二步：安装Python环境

### Windows用户

1. **下载Python**: https://www.python.org/downloads/
   - 选择 Python 3.10 或更高版本
   - ⚠️ 安装时勾选 "Add Python to PATH"

2. **验证安装**:
```bash
python --version
# 应显示: Python 3.10.x 或更高
```

### Mac/Linux用户

```bash
# Mac (使用Homebrew)
brew install python@3.10

# Ubuntu/Debian
sudo apt update
sudo apt install python3.10 python3.10-venv python3-pip
```

---

## 第三步：安装依赖

```bash
# 进入项目目录
cd D:\股票智能分析系统

# 安装所需库
pip install -r requirements.txt
```

**如果安装失败**，尝试使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 第四步：配置LLM API（重要！）

系统需要配置AI模型才能生成分析报告。支持多种LLM提供商：

### 方案一：Google Gemini（推荐新手）

**优点**: 有免费额度，注册简单，国内可访问

1. **获取API Key**:
   - 访问: https://aistudio.google.com/
   - 登录Google账号
   - 点击 "Get API Key"
   - 创建新项目或选择现有项目
   - 复制API Key

2. **配置到系统**:

   创建 `.env` 文件（或在Windows环境变量中设置）：
   ```bash
   GEMINI_API_KEY=你的API密钥
   ```

### 方案二：DeepSeek（性价比高）

**优点**: 国产模型，便宜，中文好

1. **获取API Key**:
   - 访问: https://platform.deepseek.com/
   - 注册登录
   - 进入 API Keys 页面
   - 创建新的API Key

2. **配置到系统**:
   ```bash
   DEEPSEEK_API_KEY=你的API密钥
   DEEPSEEK_BASE_URL=https://api.deepseek.com
   ```

### 方案三：OpenAI（经典选择）

**优点**: 模型强大，生态完善

1. **获取API Key**:
   - 访问: https://platform.openai.com/
   - 注册登录
   - 创建API Key

2. **配置到系统**:
   ```bash
   OPENAI_API_KEY=你的API密钥
   ```

### 方案四：Anthropic Claude

**优点**: 长文本处理能力强

1. **获取API Key**:
   - 访问: https://console.anthropic.com/
   - 注册登录
   - 创建API Key

2. **配置到系统**:
   ```bash
   ANTHROPIC_API_KEY=你的API密钥
   ```

### 方案五：使用Ollama（本地运行）

**优点**: 完全免费，数据隐私

1. **安装Ollama**: https://ollama.com/
2. **下载模型**:
   ```bash
   ollama pull qwen2.5:7b  # 或者其他模型
   ```
3. **配置到系统**:
   ```bash
   LLM_MODEL=openai/qwen2.5:7b
   OPENAI_API_BASE=http://localhost:11434/v1
   OPENAI_API_KEY=ollama  # 可以填任意值
   ```

## 第五步：配置自选股列表

编辑 `.env` 文件，添加您想分析的股票：

```bash
# 股票代码用逗号分隔
STOCK_LIST=600519,000001,300750,600036
```

常见股票代码：
```
600519  贵州茅台
000001  平安银行
300750  宁德时代
600036  招商银行
000002  万科A
600000  浦发银行
```

---

## 第六步：启动系统

### 方式一：命令行模式

```bash
# 分析指定股票
python main.py --stocks 600519

# 分析.env中配置的所有股票
python main.py

# 调试模式（查看详细日志）
python main.py --debug
```

### 方式二：Web界面模式

```bash
# 启动Web服务
python main.py --serve-only --port 8000
```

然后在浏览器访问: http://localhost:8000

### 方式三：定时任务模式

```bash
# 每天定时执行
python main.py --schedule
```

## 常见问题

### Q1: 提示 "未检测到 API Key"

**A**: 检查 `.env` 文件是否在项目根目录，并确认API Key格式正确。

### Q2: API调用失败

**A**:
1. 确认API Key有效且有足够额度
2. 如果使用Gemini，确认网络可以访问Google服务
3. 如果需要代理，设置环境变量：
   ```bash
   HTTP_PROXY=http://127.0.0.1:10809
   HTTPS_PROXY=http://127.0.0.1:10809
   ```

### Q3: 依赖安装失败

**A**:
1. 升级pip: `python -m pip install --upgrade pip`
2. 使用国内镜像源
3. 单独安装失败的包: `pip install 包名`

### Q4: 如何停止Web服务

**A**: 在运行服务的终端按 `Ctrl+C`

### Q5: 数据保存在哪里

**A**:
- 数据库: `data/` 目录
- 日志: `logs/` 目录
- 报告: `reports/` 目录
