# 系统打包和分发指南

## 重要说明：API Token 安全

### ✅ 朋友部署系统**不会**消耗您的API Token

**原因**：
1. 每个人部署系统时都需要**配置自己的API Key**
2. 您的API Key存储在您的 `.env` 文件中，**不会被分享**
3. `.env` 文件已加入 `.gitignore`，不会被提交到代码仓库

### 如何保护您的API Key

1. **不要分享**您的 `.env` 文件
2. 打包前**删除**敏感配置：
   ```bash
   # 确保删除.env文件（保留.env.example）
   rm .env
   ```
3. 确认 `.gitignore` 包含 `.env`

## 方案二：打包成压缩文件

### 优点
- 不需要Git账号
- 一次性分享
- 适合离线部署

### 步骤

#### 1. 清理不需要的文件

```bash
cd D:\Project\股票智能分析系统

# 删除虚拟环境（如果有）
rm -rf venv/
rm -rf __pycache__/
rm -rf .pytest_cache/

# 删除大文件缓存
rm -rf apps/*/node_modules/
rm -rf apps/*/.next/

# 删除敏感配置（重要！）
rm -f .env

# 删除日志和数据库（这些朋友会自己生成）
rm -rf logs/*.log
rm -rf data/*.db
rm -rf data/*.lock
```

#### 2. 创建打包脚本

**Windows (PowerShell)**:

```powershell
# 创建打包脚本
$compress = @{
    Path = "D:\Project\股票智能分析系统\*"
    CompressionPath = "D:\股票智能分析系统_完整版.zip"
    Exclude = @(
        "venv", "__pycache__", ".pytest_cache",
        "node_modules", ".next", ".git",
        "logs", "*.log", "data/*.db"
    )
}
Compress-Archive @compress
```

**Mac/Linux**:

```bash
#!/bin/bash
# 打包脚本

cd /path/to/股票智能分析系统

# 创建tar.gz压缩包
tar -czvf ../股票智能分析系统_完整版.tar.gz \
    --exclude='venv' \
    --exclude='__pycache__' \
    --exclude='.pytest_cache' \
    --exclude='node_modules' \
    --exclude='.next' \
    --exclude='.git' \
    --exclude='logs/*.log' \
    --exclude='data/*.db' \
    --exclude='.env' \
    .

echo "打包完成！"
ls -lh ../股票智能分析系统_完整版.tar.gz
```

#### 3. 分享方式

| 方式 | 适用场景 | 操作 |
|------|----------|------|
| 网盘分享 | 大文件 | 百度网盘、阿里云盘、OneDrive等 |
| 即时通讯 | 小文件 | 微信、QQ、Telegram等 |
| 邮件 | 中小文件 | 附件发送 |

## 常见问题

### Q: 朋友能看到我的历史数据吗？

**A**: 不会。历史数据存储在 `data/` 和 `logs/` 目录中，打包时已排除。朋友部署后会生成自己的数据。

### Q: 朋友可以修改源码吗？

**A**: 可以。所有源代码都包含在分发包中，朋友可以自由修改和二次开发。

### Q: 如何更新系统？

**A**:
- **Git方式**: `git pull` 获取更新
- **压缩包方式**: 重新打包并分享新版本

### Q: 朋友部署出现问题怎么办？

**A**:
1. 查看 `docs/给朋友的部署指南.md`
2. 运行 `python main.py --debug` 查看详细日志
3. 参考 `docs/DEPLOY.md` 部署文档
